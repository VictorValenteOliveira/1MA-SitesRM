O que é algoritimo?

Os algoritimos devem ser escritos em um nivel de detalhamento suficiente para que o computador consiga fazer a traducao do código para a linguagem de máquina (binaria)
O compilador precisa receber um texto formatado em uma linguagem simples, nao ambígua e precisa. Para isso as linguagens de prgramação seguem uma gramática rigida, se comparado com a lingua portuguesa, tambem seguem um vocabulário limitado, constituido por alguns poucos elementos

Exemplo:

inicio travessia da rua

olhar para direita

olhar para esquerda

se estiver vindo carro

esperar

se não

atravessar

finalizar travessia.
